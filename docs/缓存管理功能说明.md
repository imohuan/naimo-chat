# 缓存管理功能说明

## 功能概述

在设置界面中新增了缓存管理功能，可以查看和清理系统中的各种缓存数据。

## 支持的缓存目录

1. **配置备份** (CONFIG_BACKUP) - 存储配置文件的备份
2. **MCP工具调用日志** (MCP_TOOL_CALL_LOG) - 存储MCP工具调用的日志记录
3. **聊天消息** (CHAT_MESSAGE) - 存储聊天对话消息
4. **剪贴板图片** (CLIPBOARD_IMAGES) - 存储剪贴板中的图片
5. **日志文件** (LOGS) - 存储系统运行日志

## 使用方法

### 查看缓存信息

1. 打开设置界面（点击右上角的设置图标）
2. 滚动到"缓存管理"部分
3. 可以看到：
   - 总占用空间
   - 每个目录的详细信息（大小、文件数量）

### 清空缓存

1. 勾选要清空的目录（可多选）
2. 点击"清空选中目录"按钮
3. 在确认对话框中查看将要清空的目录列表
4. 点击"确认清空"完成操作

## 注意事项

- 清空操作不可恢复，请谨慎操作
- 清空后会自动刷新缓存信息
- 只能清空预定义的缓存目录，确保系统安全
